# Tlias 智能教学辅助管理系统

> 基于 Spring Boot + MyBatis 的轻量级教务管理后台系统

## 📖 项目简介
本项目是一个面向教育机构的综合信息管理平台，旨在解决传统教务管理中数据分散、统计困难的问题。系统采用前后端分离的开发模式，后端基于 Java Spring Boot 框架，实现了部门管理、员工管理、班级管理及数据统计分析等核心业务功能。

该项目严格遵循企业级开发规范，采用了标准的 MVC 三层架构设计，代码结构清晰，适合作为学习 Spring Boot 生态及 RESTful API 设计的实战案例。

## 🛠️ 技术栈
- **后端核心**: Java 8+, Spring Boot 2.x
- **持久层框架**: MyBatis / MyBatis-Plus
- **数据库**: MySQL 8.0+
- **构建工具**: Maven
- **接口文档**: Swagger / Knife4j 
- **版本控制**: Git

## ✨ 核心功能
1. **部门管理**: 支持部门的增删改查，实现组织架构的层级维护。
2. **员工管理**: 实现了员工的入职、离职、信息查询及头像上传功能；支持多条件组合分页查询。
3. **班级管理**: 维护班级基础信息，关联学生数据。
4. **数据统计**: 提供可视化的数据报表（如男女比例统计、各部门人数分布），辅助管理层决策。
5. **全局异常处理**: 统一封装了 Result 响应对象，结合 AOP 思想实现了全局异常捕获，保证接口返回格式的一致性。
